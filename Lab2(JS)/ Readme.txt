
# Описание добавленной функциональности JavaScript

## Анимации при прокрутке страницы
Для анимаций элементов при прокрутке использована библиотека AOS (Animate on Scroll). Это позволяет элементам плавно появляться в поле зрения пользователя, что делает страницу более динамичной и интересной.

### Использованные эффекты:
- `fade-up` применен к основным секциям страницы для плавного появления снизу вверх.
- `fade-right` применен к карточкам товаров, при прокрутке они плавно появляются справа налево.
- `fade-left` применен к карточкам товаров, при прокрутке они плавно появляются слева направо.


## Интерактивная форма входа
Добавлена клиентская валидация для формы входа. Валидация позволяет убедиться, что пользователь вводит корректные данные до их отправки на сервер.

### Валидация включает:
- Проверка корректности логина(латинские буквы и не менее 8 символов) .
- Проверка длины пароля (Первая буква большая, пароль должен содержать только латинские символы, также пароль должен содержать цифры(хотя бы одну) и длинной он должен бытьне менее 8 символов).

## Виджет погоды
Интегрирован виджет погоды, который отображает актуальную информацию о погоде . Виджет обращается к API OpenWeatherMap для получения данных.

### Функции виджета:
- Отображение текущей температуры и общего состояния погоды.
- Адаптивное изменение дизайна в зависимости от текущей погоды.

## Анимация фокуса на элементах формы
Элементы формы расширяются при получении фокуса, что делает интерфейс более отзывчивым и удобным для пользователя.
